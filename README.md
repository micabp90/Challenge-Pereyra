# Challenge-Pereyra

## Instrucciones para ejecutar la aplicacion:

## Backend:
en la terminal escribir:
cd backend
npm install
node index.js
Se abre en: http://localhost:3000

## Frontend:
en la terminal escribir:
cd frontend
npm install
npm run dev
se abre en:http://localhost:5173
